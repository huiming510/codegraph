需求： 代码图谱工具需支持追踪 Java 到数据库字段的完整调用链，包括 SQL 层。

问题： 当前工具只能分析 Java 代码内部的调用链，SQL 是写在字符串里的，Java 编译器不认识，因此无法把 Java 属性名（如 userId）和数据库列名（如 user_id）关联起来，导致调用链在 SQL 层断裂。

解决方案： 开发 SQL 字段映射插件，从 MyBatis XML / 注解 SQL / JPA @Query 中提取列名与 Java 属性的映射关系（如 @Result(column="user_id", property="userId") 或 user_id AS userName），自动在图谱中建立连接，补全 "Java 属性 → SQL 列 → 数据库表字段" 的完整链路。
